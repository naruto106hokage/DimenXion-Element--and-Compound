AnimationManager

What it does:
it analyzes audio at run time and changes subtitle lines according to the pauses
in audio.



Steps:

Just drag and drop AnimationSlider and SubtitleManager prefabs in hierarchy.

Make a key in smart localization root file from where the subtitle is supposed to stored.Insert the character '<' between each line breaks.
